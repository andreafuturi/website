# Andrea Futuri

This is my personal site compiled codebase (the structure is a currently work in progress Deno + Vite implementation)

## Development

Prerequisites:
- Node.js (for Vite)
- Deno (for SSR server)

If `deno` isn’t on your PATH after installing it, add this to your shell profile:

```bash
export PATH="$HOME/.deno/bin:$PATH"
```

Run dev servers:

```bash
npm start
```

Then open `http://localhost:8000`.
